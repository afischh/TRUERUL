# packages/core-language

Здесь будет жить ядро Truerul.

Сюда войдут:
- AST
- form kinds
- evaluator
- held forms space
- entity/relation/state/rule/query model
- log
- step
- later reflective operators

Именно здесь формы перестанут быть просто строками,
начнут держать собственную структуру,
а потом,
если им повезёт с воспитанием,
получат право обсуждать друг друга
без перехода к синтаксическому рукоприкладству.
