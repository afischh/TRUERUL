# packages/ansi-theme

Здесь будет жить терминальная палитра Truerul.

Нужны:
- медь и ржавчина для жара и тревоги
- холодный циан для надписей и мест
- пепельный серый для обычного текста
- тусклый жёлтый для предупреждений
- мягкий красный для сбоев и карантинных неприятностей

Никакой ярмарки RGB.
Только скромное достоинство старого кибернетического инструмента,
который уже многое видел
и поэтому не орёт цветами без нужды.
