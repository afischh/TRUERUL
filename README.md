# Truerul

**Truerul** — маленький лиспоподобный homoiconic REPL-язык
с русской surface-формой,
переключаемой на английскую,
и с quoted lispy mode как более глубокой формой речи.

Если говорить короче,
это язык не столько для описания готовых предметов,
сколько для размышления вместе с формами,
на формах,
через формы.

Он годится для сущностей,
связей,
состояний,
правил,
вопросов,
странной математики,
невозможных онтологий,
Obj0-геометрии
и прочих вещей,
которые обычно вызывают у приличных языков лёгкое дрожание синтаксиса.

## Короткая формула

Truerul строится как:

**public repo -> mobile web terminal -> PWA -> mobile app shell**

Главная поверхность v0:
**настоящая веб-консоль для телефона**
с ANSI,
prompt-ритмом,
PTY-контуром
и quoted lispy surface,
который можно переключать между RU и EN.

## Что здесь строится

- phone-first web terminal
- PTY-backed REPL session
- маленький homoiconic language core
- quoted lispy surface
- RU/EN switchable human surface
- язык для forms, entities, relations, states, rules, queries and views
- путь роста к странной математике и экспериментальным геометриям

## Чего тут пока не строится

- музей кнопок и панелей
- тяжёлый GUI
- огромная обвязка раньше ядра
- графическая ярмарка,
  притворяющаяся мышлением

Сначала — язык.
Потом — его консольное тело.
Потом — reflective operators.
Потом можно будет смотреть,
как формы начинают не только исполняться,
но и разглядывать друг друга,
что для хорошего языка всегда признак внутренней собранности,
а не праздного самолюбования.

## Базовый словарь v0

### Human surface
```text
сущность эвридика-9 :тип автономный-андроид
связь эвридика-9 ищет астрей
состояние эвридика-9 сомнение
вопрос (связь? эвридика-9 ищет X)
```

### Quoted lispy surface
```text
(entity эвридика-9 :kind автономный-андроид)
(relation эвридика-9 ищет астрей)
(state эвридика-9 сомнение)
(query (relation? эвридика-9 ищет X))
```

## Архитектурная идея

log,

step.

### packages/surface
RU/EN surface layer,
reader,
printer,
quoted forms,

форматирование,

### docs/

скриншоты,
транскрипты.

## Статус

Ранняя линия сборки.
Сейчас главная задача:
**удержать quoted lispy surface как канонический режим v0 и дать языку первое честное терминальное тело**.

Потому что язык,
который слишком рано обвешали антуражем,
часто начинает напоминать уличного пророка,
обвешанного колокольчиками: звучит бодро,
но по существу повторяет одно и то же.
