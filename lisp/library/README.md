# library/

Здесь будет жить содержательная библиотека Truerul.

Не склад украшений,
а собрание фигур,
категорий,
дихотомий,
напряжений,
парадоксов,
help-карт и цитат,
из которых машина сможет строить свои дальнейшие жесты.

Коротко:
если `src/` — это кости и кровообращение,
то `library/` — это память,
соблазн,
спор,
ирония
и долгий понятийный воздух,
без которого язык слишком быстро начинает путать стройность с жизнью.
